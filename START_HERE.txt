QUICK START (5 minutes)
═══════════════════════════════════════════════════════════════

1. Fill in .env:
   TELEGRAM_BOT_TOKEN=              (get from @BotFather)
   ANTHROPIC_API_KEY=               (get from console.anthropic.com)
   RACS_CONTACT_PHONE=+966-XX-XXXX-XXXX
   RACS_CONTACT_EMAIL=your@email.com
   RACS_CALENDLY_LINK=https://calendly.com/racs

2. Install dependencies:
   pip install -r requirements.txt

3. Validate setup:
   python validate_setup.py

4. Run the bot:
   python bot/telegram_bot.py

5. Test in Telegram:
   /ask What do I need to import electric scooters?

ARCHITECTURE (How It Works)
═══════════════════════════════════════════════════════════════

User: "What do I need to import electric scooters?"
     ↓
Bot: Extracts "electric scooter" → Queries XDS (hidden)
     ↓
Claude: Synthesizes into RACs voice
     ↓
Response: "🛴 Electric scooters require Type A certification..."
          ✓ Standards: ISO 13848-1, EN 60950
          ⏱️ Timeline: 4-8 weeks
          💰 Cost: $3K-$8K
          [CTA varies: "Ready to get started?"]
     ↓
After 3+ turns: Bot offers specialist → Captures lead → Airtable

═══════════════════════════════════════════════════════════════
KEY FEATURES
═══════════════════════════════════════════════════════════════

✓ Hidden XDS Integration
  User never sees "XDS" mentioned. Looks like RACs expertise.

✓ RACs Brand Voice
  Professional, specific (not vague), empathetic to pain points.

✓ Context-Aware CTAs
  Simple products: "Ready to get started?"
  Complex products: "Schedule a consultation"
  Urgent products: "RACs can compress timelines 30-40%"
  (Never same CTA twice per conversation)

✓ Multi-Turn Conversations
  Remembers context → answers follow-ups without repeating.

✓ Lead Capture
  After 3+ turns → "Want to connect with specialist?"
  → Collects name/email/phone → Pushes to Airtable

✓ Error Handling
  XDS down? → Graceful fallback (not error message)
  Claude timeout? → Helpful response with CTA
  All errors logged to .tmp/errors.log

SUPPORT & TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Bot not responding?
  → Check TELEGRAM_BOT_TOKEN in .env
  → Run: python validate_setup.py

Claude API errors?
  → Verify ANTHROPIC_API_KEY in .env
  → Check: console.anthropic.com for rate limits

XDS not working?
  → Run: python tools/xds_query.py
  → Check network connectivity

All errors logged to: .tmp/errors.log
